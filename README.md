# Jayne Eats (prototype)

## Software requirements

-   [Yarn](https://yarnpkg.com)
-   [Prettier](https://prettier.io/docs/en/editors.html) (for your editor)

## Running the app

```shell
# Run the app locally
yarn start

# Run tests locally
yarn test
```

## Deploying

```shell
yarn deploy
```
